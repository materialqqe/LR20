Разбиваем число FF1216 на цифры и переводим каждую цифру в десятичную систему счисления:
216 = 210 116 = 110 F16 = 1510 F16 = 1510 Складываем все цифры:

Алгоритм перевода числа из шестнадцатеричной системы счисления в десятичную (другой способ):
Умножаем каждый разряд числа на 16^n, где n - номер разряда, начиная с 0, и суммируем полученные значения:
216 * 16^0 = 2 116 * 16^1 = 16 F16 * 16^2 = 3840 F16 * 16^3 = 61440 Складываем все значения:
FF1216 = 6529810
